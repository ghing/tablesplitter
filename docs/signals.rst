=======
Signals
=======

image_split
===========

Sent when a cell image is extacted from an image containing a table

*Arguments*:

   * input_filename
   * filename
   * col
   * row

extract_image
=============

Sent when an image is extracted from a PDF

*Arguments*:

    * input_filename
    * input_md5
    * output_filename
    * output_md5
    * page
    * project
